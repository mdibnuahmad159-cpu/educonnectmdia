# **App Name**: EduConnect

## Core Features:

- Admin Authentication: Secure login for admin users with password-based authentication.
- Teacher Authentication: Teacher login using email and password credentials managed by the admin.
- Parent Authentication: Parent login using the student's ID (NIS) and password, both managed by the admin.
- User Role Management: Admin manages user roles and permissions (admin, teacher, parent) through a dedicated interface.
- Student Data Management: Admin can create, read, update, and delete student records, including NIS and password, that will be used by the parents for authentication.
- Teacher Data Management: Admin can manage teacher profiles, including assigning email addresses and setting passwords for login.

## Style Guidelines:

- Primary color: Indigo (#4B0082), conveying trust and intellect.
- Background color: Light lavender (#E6E6FA), a soft and calm tone.
- Accent color: Gold (#FFD700), to highlight important actions and information.
- Body and headline font: 'PT Sans', a modern and readable sans-serif font suitable for all text elements.
- Use simple, clear icons from a consistent set (e.g., Material Design Icons) for navigation and actions.
- Responsive layout that adapts to different screen sizes, ensuring accessibility on all devices.
- Subtle transitions and animations to enhance user experience, such as fading effects on content loading.
